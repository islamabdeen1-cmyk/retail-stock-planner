# Retail Stock Planner

Desktop application that analyzes sales and inventory data from Excel files to calculate required stock levels.

Developed by **Islam Abdeen**

## Features

- Import Excel inventory reports
- Clean product names automatically
- Group products by warehouse
- Calculate daily sales usage
- Estimate stock needed for future days
- Generate formatted Excel output

## Technologies

- Python
- Pandas
- OpenPyXL
- Tkinter
- ttkbootstrap

## Use Case

Designed for retail environments such as pharmacies or stores to help managers estimate required stock quantities based on historical sales data.
